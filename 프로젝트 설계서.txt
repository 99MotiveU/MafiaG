1. 프로젝트 개요
이 시스템은 Java를 기반으로 하여 날씨 정보에 맞는 의상 추천을 제공하는 시스템입니다. 사용자는 지역 입력 또는 위치 기반으로 날씨 정보를 요청하고, Cloth API를 사용하여 해당 날씨에 적합한 의상을 추천받습니다. 날씨 정보는 OpenWeatherMap API를 통해 가져오며, 추천된 의상은 Cloth API를 통해 제공됩니다. 추천된 의상은 DB에 저장되어 과거 이력을 조회할 수 있습니다.

2. 시스템 흐름
2.1. 사용자 입력
지역 입력 또는 위치 정보 제공: 사용자가 특정 지역(예: 서울, 부산)을 입력하거나, 위치 기반으로 날씨 정보를 요청합니다.

날씨 정보 조회: 사용자가 입력한 지역의 현재 날씨 및 예상 날씨를 OpenWeatherMap API로 조회합니다.

의상 추천 조회: OpenWeatherMap API로부터 받은 날씨 정보를 기반으로 Cloth API를 호출하여 추천 의상을 받아옵니다.

추천된 의상 표시: 날씨에 맞는 의상 추천이 Java GUI에 표시됩니다.

날씨별 의상 이력 조회: 사용자가 특정 날짜를 선택하여 과거 날씨와 추천된 의상 정보를 조회할 수 있습니다.

3. DB 설계

4. 의상 추천 API: Cloth API
4.1 API 개요
Cloth API는 날씨 정보(기온, 날씨 상태)에 맞춰 의상을 추천하는 API입니다. 이 API를 통해 사용자가 제공한 날씨 정보를 기반으로 적합한 의상을 추천받을 수 있습니다. 사용자가 입력한 지역의 날씨를 바탕으로 OpenWeatherMap API에서 날씨 정보를 가져오고, 그 데이터를 Cloth API에 전달하여 의상 추천을 받습니다.

Cloth API의 주요 기능:

기온 및 날씨 상태를 기반으로 의상 추천.

기온이 낮을 경우 더 따뜻한 의상 추천, 기온이 높은 경우 더 가벼운 의상 추천.

날씨 상태(비, 흐림, 맑음 등)에 맞는 의상 추천.

4.2 Cloth API 호출 흐름
사용자 지역 입력: 사용자가 지역을 입력하거나 위치 기반으로 날씨 정보를 요청합니다.

날씨 정보 조회: OpenWeatherMap API를 통해 해당 지역의 현재 날씨 및 예상 날씨 정보를 조회합니다.

Cloth API 호출: 날씨 정보를 기반으로 Cloth API에 요청을 보내, 날씨에 맞는 추천 의상을 받아옵니다.

의상 추천 출력: Java GUI에 의상 추천 결과를 표시합니다.

의상 추천 이력 저장: 추천된 의상과 관련된 날씨 정보를 DB에 저장하여 이후 이력 조회가 가능하도록 합니다.

5. UI 설계 (Java GUI)
Java GUI는 사용자가 쉽게 날씨 정보를 입력하고, 의상 추천 결과를 확인할 수 있도록 직관적으로 설계되어야 합니다.

5.1 주요 화면 설계
메인 화면
1)지역 입력란 : 사용자가 지역을 입력하거나 (위치 기반으로 날씨 정보를 가져올 수 있는 입력란.)

2)날씨 정보 표시 : 기온, 습도, 날씨 상태를 보여주는 화면.

3)의상 추천 표시 : 날씨에 맞는 의상 추천을 보여주는 영역.

4)날씨별 의상 이력 보기 : 사용자가 특정 날짜를 선택하여 그 날의 날씨와 추천된 의상 정보를 조회할 수 있는 기능.

5)알림 설정: 날씨 변화에 따른 의상 추천 알림을 실시간으로 제공하는 기능.

6)날씨 이력 화면: 사용자가 날짜 선택기를 통해 과거 날짜의 날씨와 추천된 의상 정보를 조회할 수 있는 기능.
		날짜를 선택하면 해당 날짜의 날씨 및 추천된 의상을 DB에서 조회하여 표시.

7)알림 기능 화면: 날씨 변화가 감지되면 알림을 통해 실시간으로 의상 추천을 제공하는 기능.	//
	예를 들어, 날씨가 추워지면 두꺼운 자켓이나 목도리를 추천하는 알림을 사용자에게 전송.	//


6. 알림 및 실시간 기능	//
알림 기능은 사용자가 실시간으로 날씨 변화에 따라 의상 추천을 받을 수 있도록 도와주는 기능입니다.

알림 설정: 사용자가 알림을 켜놓으면, 날씨 변화에 따라 적절한 의상 추천을 실시간으로 받을 수 있습니다.

날씨 변화 감지: OpenWeatherMap API에서 날씨 정보를 주기적으로 가져와 기온과 날씨 상태에 변화가 있을 때, 알림을 통해 사용자에게 의상 추천을 제공합니다.

7. 결론
Cloth API를 활용하여 기온과 날씨 상태에 맞는 의상을 추천하고, 그 결과를 Java GUI에 표시하며, DB에 추천 이력을 저장하여 과거 날씨와 의상 추천을 조회할 수 있도록 설계되었습니다. // 실시간으로 날씨 변화에 따른 의상 추천을 알림으로 제공하는 기능도 추가되어 사용자 경험을 향상시킵니다.
